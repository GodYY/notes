# Explore
Grafana 的 Dashboard UI 是关于构建可视化仪表板的。Explore 去掉了仪表板和面板选项，以便你可以专注于查询。它可以帮助您迭代，直到您有一个可以工作的查询，然后考虑构建仪表板。
